# 按钮

按钮分为三种类型：主按钮、次按钮、弱化按钮。

需要遵循三条规则：

1. 一个系统只选一种类型；

2. 次按钮二选一，绿色或灰色；

3. 主按钮在同一个操作区域最多出现一次。

## css修饰符

| 修饰符 | 说明 |
| ----- | ----- |
| xmain | 主按钮 |
| xsecondary | 铺按钮 |

| x3d | 立体效果 |
| xsmall | 小按钮 |
| xmedium | 中按钮 |
| xlarge | 大按钮 |
| xround | 圆角按钮 |
| xloading | 加载按钮 |
| xdisabled | 禁用按钮 |
